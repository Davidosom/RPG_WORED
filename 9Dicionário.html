Dicionário

Teste de ataque: É a rolagem de dados para um teste, por exemplo: 1d20, 2d20

DM: É a abreviação para Dado Mínimo, ou seja, o número mínimo para se acertar um ataque, se defender de algo,
investigar, outras coisas.

Turno: Turno é o momento em que alguém joga podendo ser o seu turno ou o turno em que o inimigo ataca/faz uma
ação.

Rodada: A rodada é quando todos jogam, incluindo jogadores e inimigos.
