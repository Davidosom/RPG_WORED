Guia das mecânicas

  • Furtividade: Quando em furtividade o jogador tem direito a 1 dado a mais para o teste de ataque.
  • Vantagem: Quando alguém recebe vantagem ela recebe 1 dado a mais para um teste de ataque.
  • Derrubado: Quando sua vida chega a 0 você é derrubado e nesse momento você tem 3 rodadas para ser mantido vivo, caso ninguém consiga ajudá-lo você terá que jogar 1d20 de constituição e se falhar você morre.
  • Morto: Quando você perde a mesma quantia de vida que você tem só que de forma negativa, (Ex: PV Max = 10, para se morto PV = -10), você morre instantâneamente.
  • Para ataques com armas brancas se usa a seguinte fórmula: Ataque = 1d20 + Lutar + Bônus de classe + Bônus de raça + Bônus de talismã + Habilidades + Habilidades da classe + outros
  • Para as armas de fogo ou de disparo é usada a seguinte fórmula: Disparo = 1d20 + Pontaria + Bônus de classe + Bônus de raça + Bônus de talismã + Habilidades + Habilidades da classe + outros
  • Para arremessar algo como pedras ou granadas é a seguinte fórmula: Arremesso = 1d20 + Arremesso + Bônus de classe + Bônus de raça + Bônus de talismã + Habilidades + Habilidades da classe + outros
