O que são níveis?

Níveis oferem ao jogador que ele fique mais forte para vencer inimigos cada vez mais poderosos, e a cada nível que o jogador avança mais difícil é
avançar para o próximo.

Qual a diferença entre cada nível?

A cada nível que o jogador avança ele tem direito a 1d4 de vida para cada ponto de constituição (por exemplo se você tem 1 ponto você joga 1d4,
mas se tiver 3 pontos você joga 3d4). E para as encarnações você obtém 1 espaço para cada nível.

Que nível eu começo?

O jogador sempre começam no nível 0 e podem chegar até o nível 12 sendo que cada classe tem diferentes tipos de buffs ao avançar de nível. A cada nível avançado o jogador ganha um poder diferente.

A seguir a lista de cada nível:

1° nível= +Con x d4 de PV + 1PH + Espaço para encarnação --> 2° nível= +Con x d4 de PV + 1PH + Espaço para encarnação -->
3° nível= +Con x d4 de PV + 1PH + Espaço para encarnação --> 4° nível= +Con x d4 de PV + 1PH + Espaço para encarnação -->
5° nível= +Con x d4 de PV + 2PH + Espaço para encarnação --> 6° nível= +Con x d4 de PV + 2PH + Espaço para encarnação -->
7° nível= +Con x d4 de PV + 2PH + Espaço para encarnação --> 8° nível= +Con x d4 de PV + 2PH + Espaço para encarnação -->
9° nível= +Con x d4 de PV + 2PH + Espaço para encarnação --> 10° nível= +Con x d6 de PV + 3PH + Espaço para encarnação -->
11° nível= +Con x d6 de PV + 3PH + Espaço para encarnação --> 12° nível= +Con x d6 de PV + 3PH + Espaço para encarnação.

Como eu aumento de nível?

Existem 3 jeitos de aumentar de nível, matando inimigos e absorvendo sua "alma"; fazendo um contrato com um demônio/anjo e treinando.

Como obter a "alma"?

Para se obter uma "alma" você deve matar o alvo e procurar em seu corpo por uma bola que varia entre as seguintes cores: branco (bondade), vermelho (maldade), roxo (neutralidade), verde (calma), azul (raiva) e preto (quantia de encarnações). A quantidade de cada uma varia para o ser vivo. Em caso de não ter alma, saiba que esse ser está com problemas mentais severos e já não possui mais vontade própria e nem ao menos sabe quem é.

Como eu consumo uma "alma"?

Para se consumir uma "alma" você precisa levar até um alquimista e dizer: "Poupe-o". Nesse momento você deve entregar a "alma" e esperar, quando ele voltar da sala ele vai entregar um copo com essa "alma" em um estado parecido com plasma.

Como fazer contrato com um demônio/anjo?

Primeiro você escolhe um anjo/demônio e desenha ou recita seu símbolo/hino para que uma forma superior apareça e então você terá que convencê-lo a te deixar mais forte. No melhor dos casos será um contrato, e no pior dos casos você morrerá.

Como funciona o treinamento?

Por padrão você não pode subir muito o seu nível, no máximo você pode sair do nível 0 para o nível 1 com muito treino. Para realmente começar você precisa achar uma academia, uma escola de encarnação ou algo do gênero para começar um treinamento.
